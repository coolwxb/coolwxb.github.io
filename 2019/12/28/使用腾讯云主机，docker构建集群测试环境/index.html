    3、cluster-master公钥分发

    在master机上执行
    ssh-keygen -t rsa
    并一路回车，完成之后会生成~/.ssh目录，目录下有id_rsa（私钥文件）和id_rsa.pub（公钥文件），再将id_rsa.pub重定向到文件authorized_keys

    文件生成之后用scp将公钥文件分发到集群slave主机

    分发完成之后测试(ssh root@cluster-slave1)是否已经可以免输入密码登录
